# Mocha-chai Automation test

example of simple Automation test API using mocha-chai. 
do ```npm install```
running the test: ```npm run test```

#### Directory structure
```
.
├── data
│   └── posts_payload.js
├── helpers
├── LICENSE
├── package.json
├── package-lock.json
├── README.md
├── requests
│   ├── posts_request.js
│   └── users_request.js
├── testcases
│   ├── posts_testcases.js
│   └── users_testcases.js
└── tests
    ├── posts_test.js
    └── users_test.js
```
